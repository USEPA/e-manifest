# e-Manifest

RCRAInfo API exposes a number of services for managing electronic and paper manifests. The following services are
available:

- [Save](./save.md)
- [Update](./update.md)
- [Delete](./delete.md)
- [Correct](./correct.md)
- [Sign](./signing.md)
- [UI link](./ui-link.md)
- [Revert](./revert.md)
